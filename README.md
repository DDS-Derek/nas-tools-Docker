# nas-tools-Docker

使用方法与官方镜像一致

### 全功能版本

|       架构       | 支持情况 |  标签   |
| :--------------: | :------: | :-----: |
| amd64 \| x86-64  |    ✅     |  `<Version>-amd64`  |
| arm64v8 \| arm64 |    ✅     | `<Version>-arm64v8` |
|  armv7 \| armhf  |    ✅     |  `<Version>-armv7`  |

总标签：`docker pull ddsderek/nas-tools:latest` `docker pull ddsderek/nas-tools:<Version>`
